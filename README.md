# Five Nights At Hamber (FNAH)

<b>IMPORTANT:</b> <a href='https://www.python.org/downloads/'>Python 3.14</a> as well as <a href='https://www.pygame.org/download.shtml'>Pygame</a> must be installed on the device for this program to run.

Developed by vhgj and harperezwin.

<h2>How to start the game</h2>
1. To start the game, you must use your computer's terminal and navigate to the location of the game files.
<h3>Windows:</h3>
I'm ngl i don't know much abt windows, but i know if you go into the folder and right click you can 'open in terminal'. Do that. Unless you know a better way. Feel free to tell me.

<h3>Linux/Mac:</h3>
Open the terminal and use `cd ~/FILEPATH/`<br>
Example:<br>
`cd ~/Downloads/fivenightsathamber` or `cd ~/myFolder/mySubFolder/fivenightsathamber`
<br><br>
2. Once you're in the folder, type in `python3 fnah_stable.py` (recommended) or `python3 fnah_new.py`
